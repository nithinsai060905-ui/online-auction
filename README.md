# Online Auction/Bidding System

## Description
A web-based auction system where users can place bids on products. The system tracks the highest bid and declares a winner when the auction ends.

## Technologies Used
- HTML
- CSS
- JavaScript

## Features
- Multiple Auction Items
- Bid Validation
- Countdown Timer
- Bid History
- Winner Declaration
- Local Storage

## How to Run
1. Download project files
2. Open index.html in browser
3. Start bidding
